## 1. Arah Visual
`Nine Studio` memakai pendekatan visual dark-premium yang tegas, modern, dan terasa seperti studio teknologi yang serius mengerjakan produk bernilai tinggi. Logo merah-hitam menjadi pusat identitas, lalu diterjemahkan ke UI lewat kontras tinggi, panel gelap bertingkat, garis aksen merah, dan komposisi layout yang rapi namun berkarakter.

## 2. Identitas Brand
- Kata kunci: tegas, teknikal, premium, cepat, presisi, trusted.
- Nuansa: agency teknologi boutique dengan kemampuan web, mobile, automation, dan AI.
- Kesan utama yang harus terasa: `siap produksi`, `rapi`, `profesional`, dan `mudah diajak deal`.

## 3. Sistem Warna
- `Brand Red`: `#F20D1A`
- `Brand Red Deep`: `#B50812`
- `Carbon Black`: `#070707`
- `Soft Black`: `#111111`
- `Steel Gray`: `#6F7278`
- `Cloud White`: `#F3F4F6`
- `Border Dark`: `rgba(255,255,255,0.1)`
- `Glow Red`: `rgba(242,13,26,0.28)`

Penggunaan:
- Background utama memakai `Carbon Black`.
- Panel/kartu memakai `Soft Black` dengan border halus.
- CTA utama memakai `Brand Red`.
- Text utama memakai putih terang dengan secondary text abu lembut.

## 4. Tipografi
- Heading display: font techno-modern yang kuat untuk headline hero dan section title.
- Body text: sans-serif yang bersih untuk keterbacaan tinggi.
- Hierarki:
  - Hero title sangat dominan.
  - Section title medium-bold.
  - Body text ringkas, tidak panjang, fokus pada manfaat dan hasil.

## 5. Tata Letak
- Model satu halaman dengan section anchor.
- Topbar sticky transparan gelap dengan blur halus.
- Hero dibuat dua kolom: copywriting kiri, visual atau preview kanan.
- Layanan, portfolio, dan pricing menggunakan grid cards dengan ritme yang konsisten.
- CTA penting muncul berulang di hero, pricing, dan contact.
- Footer ringkas tetapi tetap menampilkan navigasi cepat dan identitas brand.

## 6. Komponen Utama
| Komponen | Arah Desain |
|----------|-------------|
| Topbar | Logo kiri, menu tengah/kanan, tombol CTA merah, saat scroll berubah lebih solid |
| Hero | Headline besar, subheadline konversi, dua CTA, badge capability, metrics singkat |
| Service Card | Ikon lucide, judul tegas, deskripsi singkat, hover naik tipis |
| Portfolio Card | Preview besar, label kategori, hasil proyek, tag stack, overlay elegan |
| Pricing Card | Fokus pada keterbacaan, paket populer diberi highlight merah dan glow lembut |
| Checkout Panel | Panel gelap elevated, step jelas, input rapi, summary fixed, tombol bayar dominan |
| Admin UI | Dashboard utility-first, lebih sederhana dari website publik tetapi tetap konsisten brand |

## 7. Interaksi dan Motion
- Reveal halus saat section masuk viewport.
- Hover state pada kartu: translate ringan, border glow merah, shadow lembut.
- Tombol CTA punya transisi warna dan shadow.
- Angka statistik dan badge dapat dianimasikan ringan saat load.
- Hindari animasi berlebihan agar tetap terasa premium dan cepat.

## 9. Catatan Implementasi UI
- Desain mengikuti referensi yang Anda lampirkan, tetapi dibuat lebih rapi, lebih premium, dan lebih siap dipakai untuk konversi.
- Semua gambar visual website publik sebaiknya konsisten dalam tone merah-hitam dan dunia kerja developer modern.
- Pricing dan portfolio harus dibangun dari data admin agar mudah diubah tanpa edit kode.
- Checkout perlu terasa singkat: pilih paket, isi brief, bayar, selesai.
